📱 Event Access Manager - Fatec Academic Project
Event Access Manager é um aplicativo Android desenvolvido em Java no Android Studio, utilizando o Firebase como banco de dados em tempo real. O projeto foi desenvolvido como parte de um trabalho acadêmico na Fatec e tem como objetivo facilitar a gestão de acesso em eventos, permitindo que usuários criem e participem de eventos, além de gerar certificados de participação.

✨ Funcionalidades
👤 Para Usuários Comuns
Cadastro e login com autenticação via Firebase

Criar eventos com nome, descrição, data e local

Participar de eventos criados por outros usuários

Visualizar histórico de eventos que participou

🎟️ Para Organizadores de Eventos
Controle de acesso em tempo real (registro de entrada e saída de participantes)

Geração de certificados (opcional, em formato PDF) para os participantes

Lista de presença com horários de check-in e check-out

🛠️ Tecnologias Utilizadas
Linguagem: Java (Android SDK)

Banco de Dados: Firebase Realtime Database & Firebase Authentication

Ferramenta de Desenvolvimento: Android Studio

Geração de Certificados: iTextPDF ou similar (para exportar PDFs)
